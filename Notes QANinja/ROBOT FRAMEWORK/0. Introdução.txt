O QUE SÃO MICROSERVIÇOS
-------------------------
	Arquitetura de softwares independentes normalmente dividos por unidade de negócio desenvolvidas de forma MICRO, ou seja, (um aplicativo para resolver cada tipo de problema - exemplo uma API responsável pelo cadastro de fornecedores e outra API responsável pelo cadastro de entregadores)
	Os microserviços podem se comunicar entre si. Para o usuário do Software ele é uma coisa "unica", porem sua arquitetura foi feita separamente modularizada por microserviços.

ROBOT FRAMEWORK
------------------
	Framework de automação de testes de código aberto e genérico. Ou seja, como ele é genérico, é possivel testar softwares de linguagens diferentes (inserindo bibliotecas) - Web, mobile, API, desktop....

FERRAMENTAS
---------------
	Para realizar o curso baixar as seguintes ferramentas:
	-- PYTHON 3.9.9 (https://www.python.org/downloads/release/python-399/)
	-- PIP 21.2.4 Gereciador de pacotes do python, por onde conseguimos fazer a instalação de bibliotecas inclusive do ROBOT. (A instalação do mesmo é feita automaticamente ao instalar o PYTHON)
	-- ROBOT FRAMEWORK Necessário seguir o passo a passo descrito em (https://robotframework.org/?tab=1#getting-started)
	-- VISUAL STUDIO CODE (https://code.visualstudio.com/)
	-- GIT BASH (https://gitforwindows.org/)
	-- NOTE JS pegar a versão LTS (nodejs.org/en) onde microserviço curso foi desenvolvido

EXTENSÕES VSCODE
-----------------
Abra o VSCODE para fazer as seguintes configurações
	ROBOT FRAMEWORK INTELLISENSE (framework de teste que vamos usar)
	HYPER TERM THEME (Tema para terminal de interface de linha de comandos)
	MATERIAL ICON THEME (Muda os icones padrões do VSCode)

Fazendo integração do GIT como o HAYPER.
	Acesse o site (https://hyper.is/) e faça o download do instalador e instale.
	Inicie o programa HAYPER, acesse as configurações > edit > preferences
	Sera aberto notepad com arquivo de configurações

Altere os parametros SHELL para: shell: 'C:\\Program Files\\Git\\git-cmd.exe',
Altere os parametros SHELLARGS para: shellArgs: ['--command=usr/bin/bash.exe', '-l', '-i'],
Altere os parametros ENV para: env: {'TERM':'cygwin'},
Reinicie o computadaor e volte no site do HAYPER e instale os Plugins caso desejado.
